# Global Theme Requirements Checklist

When building **any** new theme for this portfolio, this checklist must be referenced to ensure feature parity and consistency across all designs. Every theme MUST implement creative, thematic ways to display the following data and interactive elements:

## Design & Layout Guidelines
- [ ] **Scrolling Layout:** The layout should feel like a substantial, robust website. Ensure the content area is large enough to allow the user to naturally scroll down to discover information, rather than aggressively cramming everything into a single fixed viewport.

## Core Data Sections
- [ ] 1. Who I am (Name & Role)
- [ ] 2. My background (Bio)
- [ ] 3. Skills (Categorized list of technical/soft skills)
- [ ] 4. Projects (Featured work)
- [ ] 5. Certifications (Credentials and achievements)
- [ ] 6. Education (Degrees, universities, dates)
- [ ] 7. Resume (Downloadable file)
- [ ] 8. Contact information (Email, phone, location)
- [ ] 9. Social links (LinkedIn, GitHub, Twitter, etc.)
- [ ] 10. Timeline (Chronological ordering of experience/education)
- [ ] 11. Fun interactive sections (Unique thematic elements, games, or easter eggs)

## Explicit Interactive Requirements
To ensure the user can actually interact with the data, these specific buttons and actions **must** be implemented:

- [ ] **Projects:** Must include a clear `Play Demo` / `Live Demo` button (linked to `liveUrl`).
- [ ] **Projects:** Must include a `Source Code` / `GitHub` button (linked to `githubUrl`).
- [ ] **Certifications:** Must include a `View Credential` button or link (linked to `url`).
- [ ] **Certifications:** Must include a `Download` icon/button (linked to `fileUrl`).
- [ ] **Resume:** Must include a prominent `Download Resume` / `Download Dossier` button.
